Caesar Cipher


The Caesar Cipher is a simple substitution cipher that shifts the letters of the alphabet by a fixed number of positions. Click the link above to learn more and practice using the Caesar Cipher.



Monoalphabetic Cipher


The Monoalphabetic Cipher is a substitution cipher that replaces each letter in the plaintext with a different letter from the alphabet based on a given key. Click the link above to learn more and practice using the Monoalphabetic Cipher.
